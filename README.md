# Hand Volume Controller 🎚️✋

A computer vision project that controls system volume using hand gestures using OpenCV and MediaPipe.

## 🚀 Features
- Real-time hand detection
- Volume control using finger distance
- Live volume bar
- Works with webcam

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- PyCaw
- NumPy

## 📁 Files
- `main.py` – Main program
- `test_cam.py` – Camera test script
- `requirements.txt` – Dependencies

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
